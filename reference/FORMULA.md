# 電話號碼五格數字學 — 公式解碼(來自 reference/原始公式.xlsx)

這份是從 `原始公式.xlsx` 的「電話選號計算 / 電話選號」分頁解碼出來的核心公式,
作為實作的**權威依據**。原 xlsx 因為是「空白範本」有大量 `#REF!`(個人資料庫分頁被移除),
但電話計算的核心邏輯不依賴那些 #REF!,可獨立實作。

> 實作時仍須開啟 `reference/原始公式.xlsx` 的「電話選號計算」「電話選號」「比對」分頁,
> 補齊本文件標記為【待從 xlsx 補齊】的對照表(五行生克吉凶、各格意義文字)。

## 一、輸入與分組
- 使用者輸入一組電話號碼(數字)。
- 號碼切成 **3 組**(xlsx 範例:第1組=`036`、第2組=`102`、第3組=`682`)。
- 分組規則【待從 xlsx/使用者確認】:範例為 3-3-3=9 碼。台灣手機為 10 碼(09xx-xxx-xxx),
  實作時分組規則設成**可設定**,並提供台灣手機的預設切法(待確認:去掉開頭 0 後 3-3-3,或 4-3-3)。

## 二、數值轉換
- 每個數字位的值 = 該數字本身(0→0, 1→1, …, 9→9)。
  (xlsx 用 VLOOKUP a0..a9 對應 0..9;字母 a..z 的對應是給英文姓名用的,電話只用數字。)
- 每組加總 = 該組所有數字相加。
  - N1 = 第1組數字和、N2 = 第2組數字和、N3 = 第3組數字和。

## 三、五格(核心)
| 格 | 公式 | 影響權重 |
|----|------|---------|
| 總格(本體) | N1 + N2 + N3 | 50% |
| 天格(父系) | N1 + 1 | 5% |
| 人格(官祿) | N1 + N2 | 15% |
| 地格(母系) | N2 + N3 | 10% |
| 外格(財官) | N3 + 1 | 25% |

## 四、五格 → 五行
取每個格的**個位數**(最後一位),對應五行:
- 1,2 → 木   3,4 → 火   5,6 → 土   7,8 → 金   9,0 → 水

## 五、延伸關係格(取兩格相加再取個位 → 五行)
- 子息 = 天格 + 外格
- 健康 = 外格 + 地格
- 配偶 = 天格 + 人格
- 朋友 = 人格 + 地格
（xlsx 另有 父系/官祿/財官/母系 等,對應上面的天/人/外/地格本身。）

## 六、吉凶判斷(已解碼,資料已備妥)
吉凶來自兩套系統,兩者都要呈現:

### (A) 81 數理(每個格的數值 → 吉凶 + 斷語)
- 完整對照表已抽出存成 **`reference/81數理.json`**(81 條,老師原版斷語),格式:
  `{ "28": { "symbol":"X", "luck":"凶", "text":"主遭難別離之運…" }, ... }`
  - 符號:`○`=吉、`▲`=半吉(吉帶凶)、`X`=凶。
- 五格的每個數值(總格/天格/人格/地格/外格)直接查這張表 → 得吉凶符號 + 斷語。
- 數值 > 81 的處理:本系統各格和通常 ≤ 81;若超過,用 81 數理慣例 `((n-1) % 80) + 1` 映射回 1–81。
- 各格影響權重(用於整體評分):總格 50%、外格 25%、人格 15%、地格 10%、天格 5%(見「電話選號」C 欄)。

### (B) 五行生克(本體 與 各關係格 之間)
- 本體 = 總格的五行。各「關係格」取其數值的個位 → 五行,再看與本體的生克關係:
  | 關係格 | 數值來源 |
  |--------|---------|
  | 子息 | 天格 + 外格 |
  | 健康 | 外格 + 地格 |
  | 配偶 | 天格 + 人格 |
  | 朋友 | 人格 + 地格 |
  | 官祿 | = 人格 | 財官 = 外格 | 父系 = 天格 | 母系 = 地格 |
- 生克五種關係與解讀基調(取自 xlsx「電話選號」I 欄範本,實作可照此語氣):
  - **比和**(同五行):平輩朋友、無分長幼;與行運相關(運好則旺)。
  - **本體生 X**(本體付出):盡心效忠/孝敬,卻未必得到等值回應(付出型)。
  - **本體剋 X**(本體主導):有約束力、須努力經營但必有所獲(主導型,多為吉)。
  - **X 生本體**(得助):受益,如「健康宮生本體 → 身體少病痛」。
  - **X 剋本體**(受制):較不利,須留意。
- 生克標準規則(可直接寫死):木生火、火生土、土生金、金生水、水生木;木剋土、土剋水、水剋火、火剋金、金剋木。

### (C)「好號碼」定義(階段二篩選用)
- 重點格(總格 50%、外格 25%)為 `○`(吉)→ 必要條件。
- 整體加權吉凶分數高(把各格 ○/▲/X 給分後依權重加總)。
- 五行關係以相生/比和為主、無關鍵相剋。
- 實作:數理與生克判定**忠於 81數理.json 與上述規則**;給使用者看的整體解讀文字可另由 LLM adapter 生成更口語版本(使用者同意:資料為準 + AI 補充)。

## 七、驗證用範例(請寫成單元測試)
輸入三組:第1組=036、第2組=102、第3組=682
- N1 = 0+3+6 = 9
- N2 = 1+0+2 = 3
- N3 = 6+8+2 = 16
- 總格 = 9+3+16 = 28 → 個位 8 → **金**
- 天格 = 9+1 = 10 → 個位 0 → **水**
- 人格 = 9+3 = 12 → 個位 2 → **木**
- 地格 = 3+16 = 19 → 個位 9 → **水**
- 外格 = 16+1 = 17 → 個位 7 → **金**

計算引擎必須對上述輸入產出上述五格數值與五行,作為回歸測試的黃金案例。
